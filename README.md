# 📊 Business Sales Dashboard

An interactive **Sales Analysis & Forecasting Dashboard** built using Streamlit.  
This application provides real-time insights, visual analytics, and business forecasting tools.

---

## 🚀 Features

🔐 **Authentication System**
- Login & Signup (SQLite database)
- Secure password hashing

📊 **Sales Analysis**
- Region-wise sales & profit
- Monthly trends
- Product contribution analysis
- Sales vs Units scatter plot

📈 **Advanced Insights**
- Correlation heatmap between key metrics

🔮 **Business Forecasting**
- Profit prediction
- Demand forecasting
- What-if analysis (price simulation)

🎨 **Modern UI**
- Dark theme dashboard
- Interactive charts (Plotly)
- Clean and professional layout

---

## 🛠 Tech Stack

- **Frontend**: Streamlit  
- **Backend**: Python  
- **Database**: SQLite  
- **Visualization**: Plotly  
- **Libraries**: Pandas, PIL
